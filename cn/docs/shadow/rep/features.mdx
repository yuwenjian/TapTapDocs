## 功能介绍

TapTap 跳转 SDK 旨在服务在 REP 平台做置换的游戏，提供游戏客户端到 TapTap 客户端的直接跳转服务。

![](https://img.tapimg.com/market/images/cf461f39967d2c15d9e4a85e314c588f.png)

- 点此前往 [REP 资源置换平台](https://rep.taptap.cn/)
- 点此了解 [如何获取 REP 短链](https://rep.taptap.cn/docs/taprep-game-jump#%E6%AD%A5%E9%AA%A41:%E7%94%9F%E6%88%90%E7%9B%91%E6%B5%8B%E9%93%BE%E6%8E%A5)

## 服务优势

### 流程短，效率高
减少使用链接额外唤起浏览器的步骤，提升用户体验。

### 接入便捷，成本低廉
封装 TapTap 唤端逻辑，多种场景一步到位。

## 注意事项

- 请确保填入 TapTap 跳转 SDK 的链接为 REP 后台生成的链接，如非 REP 生成链接则可能导致跳转失败。
- 接入跳转 SDK 后，当用户没有 TapTap 客户端时，默认用浏览器打开填入 SDK 的跳转链接。
