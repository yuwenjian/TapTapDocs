## 功能概述
TapTap 好友功能支持在游戏内引入 TapTap 好友关系，帮助游戏社交冷启动。
支持查看好友信息，发送邀请上线、邀请组队消息，让好友一键打开游戏、进入指定组队/房间页面。含聊天版还支持与好友跨端聊天。

### 好友（含聊天）与好友（不含聊天）的主要区别
| 功能                            | 好友（含聊天） | 好友（不含聊天） |
| ------------------------------- | ------ | ------------------------------ |
| 好友的使用方式            | SDK 自带好友聊天一体界面，无需自行绘制好友、聊天界面 | 游戏获取好友数据，自行绘制界面、处理展示逻辑 |
| 是否包含聊天能力                   | 是      | 否 |
| 是否需要额外请求用户授权好友关系     | 否      | 是 |

建议选择其中一个版本接入即可。游戏如有特殊需要，也可同时使用好两个版本。

## 服务优势
- 快速建立游戏关系，支撑游戏玩法
    - 直接利用玩家在 TapTap 已结成的社交关系导入游戏，配合邀请消息，辅助组队、联机邀请等玩法。
- 便捷分享游戏，拉新促活
    - 邀请消息可发送给未玩本游戏的 TapTap 好友，对方通过邀请消息可一键下载/启动游戏，社交传播带动增长与活跃。
- 单机游戏无需额外备案，无需服务器成本（不含聊天版不支持）
    - 不需要联网游戏备案的单机游戏，接入好友后仍旧无需备案，无需承担服务器成本。
- 接入简单，功能全面（不含聊天版不支持）
    - SDK 封装了好友管理与聊天全套轻量界面，横竖屏、深浅色模式齐备，无需游戏处理复杂的界面与逻辑，也避免了向玩家申请好友关系授权的步骤。

## 准备工作
- 接入 [TapTap 登录](https://developer.taptap.cn/docs/sdk/taptap-login/features/) 
    - 玩家在使用 TapTap 好友的能力前，需在游戏中完成 TapTap 登录绑定，将依据所登录账号，返回对应好友数据。
- 开启 TapTap 好友服务
    - 在开发者中心游戏的控制台，通过「游戏服务 - TapTap 好友」开启。

## 应用场景示例
:::info
请注意，TapTap 用户「互相关注即为好友」。无论在游戏还是 TapTap 客户端中，被关注后会收到提醒，完成回关即成为好友。相对的，取关即解除好友关系。

:::
### SDK 自带好友与聊天界面（含聊天版适用）

![](https://img.tapimg.com/market/images/1d0775180bfcb691df3797fdb8262772.png)

承载所有好友，及与好友的聊天
- 游戏中绘制好友、聊天入口，点击调起好友聊天一体界面，可进行好友管理与聊天。
- 聊天入口处通过获取未读消息数，绘制红点或数字，引导玩家查看。

### 自行绘制好友列表（不含聊天版适用）

展示 TapTap 好友列表
- 游戏可使用 TapTap 好友数据，自行绘制游戏好友界面。并在好友列表灵活绘制诸如「赠送友情点」「送礼」等功能按钮，实现好友相关的游戏系统功能。
- 若游戏自己的好友系统有多种加好友来源，可将 TapTap 好友作为加游戏好友的来源之一，绘制 TapTap 好友列表，并附加「加为游戏好友」按钮。

### 添加好友

![](https://img.tapimg.com/market/images/19dcfe47dfdaac0c5584f94fec387b57.png)

在游戏中实现新好友的发现与添加。
- 在游戏中合适位置（如在排行榜、组队战斗结算页面中，点击玩家头像、个人主页的查看资料菜单）绘制 TapTap 名片入口，点击调起对应弹窗。
    - TapTap 名片入口按钮建议表现形式为：加好友、名片、个人资料。便于玩家理解此处可查看其 TapTap 信息并加好友。
- 通过 TapTap 名片完成关注。
- 有新的粉丝时会收到提醒，在 SDK 提供的好友聊天界面中完成回关即成为好友。若为不含聊天版，则需要在 TapTap 客户端中完成回关。

### 邀请游戏/组队

![](https://img.tapimg.com/market/images/8c0ce5ff7e9d3cac077337a0d31ceb44.png)

在游戏中向 TapTap 好友（无论是否在线，无论是否玩过当前游戏）发送邀请消息，好友点击消息即可一键拉起游戏，甚至直接进入指定队伍/房间。
- 游戏中绘制邀请入口，点击调起 SDK 选择好友弹窗完成邀请。
- 被邀请者在 TapTap 看到邀请，点击一键拉起游戏、进入队伍/房间。
