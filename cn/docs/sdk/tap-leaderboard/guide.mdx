### й…ҚзҪ® FileProvider

еңЁ Android е№іеҸ°пјҢдёәдәҶжӯЈзЎ®иҺ·еҸ–еҲҶдә«еӣҫзүҮзҡ„и·Ҝеҫ„пјҢйңҖиҰҒй…ҚзҪ® FileProviderгҖӮ

1. еңЁ `AndroidManifest.xml` дёӯж·»еҠ  FileProvider й…ҚзҪ®пјҡ

```xml
<provider
    android:name="androidx.core.content.FileProvider"
    android:authorities="${applicationId}.fileprovider"
    android:exported="false"
    android:grantUriPermissions="true">
    <meta-data
        android:name="android.support.FILE_PROVIDER_PATHS"
        android:resource="@xml/file_paths" />
</provider>
```

2. еңЁ `res/xml` зӣ®еҪ•дёӢеҲӣе»ә `file_paths.xml` ж–Үд»¶пјҲеҰӮжһңдёҚеӯҳеңЁеҲҷеҲӣе»әпјүпјҢеҶ…е®№еҰӮдёӢпјҡ

```xml
<?xml version="1.0" encoding="utf-8"?>
<paths>
    <!-- е…Ғи®ёи®ҝй—®еә”з”Ёз§Ғжңүзӣ®еҪ•дёӢзҡ„ files/shared_images/ -->
    <files-path name="shared_images" path="shared_images/" />
</paths>
```

### и®ҫзҪ®еҲҶдә«еӣһи°ғ

```java
import com.taptap.sdk.leaderboard.androidx.TapLeaderboard;

    public void onShareSuccess(String localPath) {
```
