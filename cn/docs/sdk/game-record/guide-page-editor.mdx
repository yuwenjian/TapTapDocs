## 角色页介绍

一张角色页由多个模块构成。开发者可以在后台自主选择所需的角色信息模块，利用我们提供的拼图式角色信息模块，构建出理想的角色页样式。

我们拆解每个模块的展示位（即组件，图示中用数字标出），每个组件支持开发者选择是否展示。对于需要展示的组件，我们提供该组件的常用预设内容。

## 配置教程

### 选择角色查询模版

**选择预设模板**

我们提供四类常用角色查询模板（根据主流线上游戏角色查询提供），请根据游戏类型选择：
- 模板仅预配置角色信息模块，不影响 UI 样式
- 支持随时更换模板，但是切换后您配置的数据将被清空

**模块自由调整**

- 选定模板后您可以删除默认模块或添加新模块，支持实现完全自定义的角色页结构
- 您可以在页面右侧实时预览您目前搭建的角色页结构

### 配置模块内容

**设置模块展示的信息**

- **模块名称**：根据您的需要配置名称，名称将作为模块标题在角色页内展示
- **模块组件**
  - **接口字段**：您在此配置的内容将直接影响后续导出的接口文档
    - 优先选择预设选项：无需精准匹配，例如：选择"普通资产数"对应游戏内的"金币"数据
    - 自定义字段：如预设选项不满足需求，可删除当前选中项后手动输入
      - 格式要求：仅可使用小写字母、数字、英文下划线
      - 示例：`玩家首次登录日` 可填写 `first_login_date`
  - **数据名称**：请用游戏内实际名称命名，方便后续查询，部分名称将在角色页内展示
- **图片素材的上传**：如果您选择了图片类型组件，您需要将图片素材**以接口传输的字段数据命名**后进行上传，我们会根据您接口推送的字段内容，匹配素材库内的图片

**快速定位调整**

您可以通过点击实时预览内容，快速定位到您希望调整的模块/组件。

### 设置页面样式

**页面配色**

- **平台默认样式**：平台提供 7 套不同配色的设计方案，选择主题色即可快速完成样式设置
- **定制化样式**：支持您设置每个文本的色值、模块间距以及模块背景图，您可以通过右侧实时预览您配置的样式

**页面头图**

- **选择"默认"**：将自动复用游戏论坛顶部宣传图作为角色页头图，头图内容会随宣传图更新同步刷新
- **选择"自定义"**：选择此项可手动上传头图，上传后图片固定显示，不受宣传图更新影响
