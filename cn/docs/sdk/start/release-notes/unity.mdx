<h4> 修复 </h4>

- 修复在部分 windows 设备下 editor 模式运行闪退问题
- 登录模块：修复部分网络异常情况下未触发回调的问题

<h2> 4.5.2 </h2>

更新时间 2024-12-23

<h4> 优化 </h4>

- [PC] 在对外 API 添加初始化参数及应用配置信息校验提示
- TapTap 登录：优化登录流程，当登录未结束时，再次发起登录会返回登录失败的回调

<h4> 修复 </h4>

- [Android] 合规认证：修复当设置的用户标识与数据分析不一致时，部分情况下可能会收到退出回调的问题
- [iOS] 合规认证：修复当同时集成数据分析时，部分情况下可能会收到退出回调的问题
- [PC] 数据分析：修复使用 SetUserID(userId, properties) 接口且 properties 为 null 时，设置账号 ID 失败的问题


<h2> 4.5.1 </h2>

- [Android] 优化混淆配置

<h4> 修复 </h4>

- TapTap 登录：修复用户信息中 unionId 字段命名与文档不一致问题
- [PC] 修复 Editor 模式下运行异常及部分设备兼容性问题
- [Android] 内购服务：修复偶现支付成功状态不消失的问题
- [Android & PC] 成就系统：修复连续调用分步成就增长接口时重复上报的问题


<h2> 4.5.0 </h2>

更新时间 2024-11-18

<h4> 功能 </h4>

- [Android] 新增 [ 内购服务 ] ：提供整合商品管理、支付渠道、订单管理的便捷内购解决方案


<h4> 优化 </h4>

- 合规认证：当 Startup 接口异步处理未结束时，如果重复调用该接口，SDK 将直接返回
- 删除冗余 Protobuf 依赖
- [iOS] 内嵌动态：添加隐私说明文件，删除在 framework 中冗余的资源文件
- [PC] TapTap 登录：更新内部网络库及登录页面扫码引导图

<h4> 修复 </h4>

- [Android] 修复 OAID 参数设置无效问题
- [Android] 数据分析：修复调用仅包含用户 ID 的 SetUserID 接口时的异常问题
