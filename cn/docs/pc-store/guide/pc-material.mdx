- 尺寸：512 x 512px  
- 格式：png/jpg  
- 要求：请直接上传直角方图图标，请勿自行裁切圆角

### 2. 电脑/主机游戏封面图

- 尺寸：460 x 215px  
- 格式：只支持 png/jpg  
- 运营建议：使用游戏主要角色、主要画面场景作为图片的核心视觉

### 3. PC版 游戏logo

- 尺寸：游戏 LOGO 素材尺寸需满足「1280px宽」X「720px高」

![gameadmin](https://img.tapimg.com/market/images/ec64b59f62304cb7ba6d147871d47980.png)

### 5. 游戏库背景壁纸

- 尺寸：1920 x 1080px  
- 格式：png/jpg  
- 图片大小：3 MB以内  
- 要求：避免出现游戏LOGO、宣传标题等任何形式的文字内容  
- 建议：保证图片质量高清且精美，注意页面左侧可能出现的公告与活动UI，避免遮挡主体角色或重要元素

## 二、通用物料

### 1. 文案

游戏如果需要上架中国大陆地区，请提供简体中文版文案与物料。  
文案里请勿包含兑换码（包括通码）、玩家交流群信息、除 Steam 平台和 TapTap 详情页链接之外的外部链接。

#### 1.1 简介 (*必填)

- 内容：描述游戏的类型、玩法以及特色  
- 运营建议：建议将主要内容控制在前 50 个字，有利于加深用户对游戏的第一印象  

#### 1.2 开发者的话

- 内容：描述游戏设计理念、游戏开发心路历程  
- 运营建议：建议开发者的话具备个人特色且真情实感，并建议将主要内容控制在前 50 个字  

**提示**：为避免影响用户端展示效果，简介与开发者的话请勿雷同。

#### 1.3 更新日志

- 内容：描述本次游戏更新内容与游戏版本变动，不得出现宣传、广告、联系方式或其他无关内容  
- 运营建议：建议文案简洁明了，使用有序列表展示更新点，比如：1. 2. 3. …

![gameadmin](https://img.tapimg.com/market/images/812df92c13bccd980c685d976fc0a06e.png)

---

### 2. 实机录屏及游戏截图

#### 2.1 游戏实机录屏 (*可下载或可试玩必填)

- 格式：MP4  
- 大小：500 MB 以内  
- 长度：视频长度建议在 2 分钟内，最长不得超过 3 分钟  
- 内容：视频需演示游戏的核心玩法  

**提示**：  
- 请在前 5s 内展示核心玩法，避免使用抽卡动画/过场动画等类似PV画面占据主要时长。  
- 请勿只展示角色、立绘、Loading界面等非玩法内容。  
- 请勿添加游戏本体以外的配音和字幕。  
- 前30秒内请勿全屏展示Logo。  
- 请勿使用画中画或拍出游玩设备。  
- 片尾请勿含有营销诱导性文案。  
- 请提供高清视频。

![gameadmin](https://img.tapimg.com/market/images/d668e069652163b96fa06446300c7b56.png)

#### 2.2 截图 (*必填)

- 图片方向：支持横图或竖图，但请保证每张截图的图片方向一致  
- 尺寸：横版 1280 x 720px 及以上；竖版 720 x 1280px 及以上  
- 格式：png/jpg  
- 数量：至少 3 张，请勿上传相同截图  
- 大小：建议每张截图 1 MB以内  

**提示**：  
- 必须有实机核心玩法内容展示，实机画面至少占图片的 50%。  
- 概念类、广告宣传类图片至多 2 张。  
- 请勿使用过度美化或误导性截图。

![gameadmin](https://img.tapimg.com/market/images/9449bd4adc323b33b6b3ae494a6090ef.png)

### 3. 推广物料

#### 3.1 宣传图 16:9 (*必填)

- 用途：首页推荐、今日游戏等卡片展示页面，也可作为视频封面  
- 尺寸：1920 x 1080px（16:9）  
- 格式：png/jpg  
- 内容：使用游戏主要角色、主要画面场景作为核心视觉，必须展示清晰可见的游戏名  

**提示**：  
- 除了游戏名外，请勿使用其他文本。  
- 请勿将主要元素放置在可能被裁剪的区域。  
- 请勿使用纯色底/渐变色仅展示标题。

![gameadmin](https://img.tapimg.com/market/images/1ec50b2b6616a7037d86bf01ec6a9dc3.png)

#### 3.2 宣传图 1:1

- 用途：竖版实机录屏的视频封面  
- 尺寸：1440 x 1440px（1:1）  
- 格式：png/jpg  
- 内容：主体内容应与宣传图 16:9 保持一致，仅调整尺寸和布局细节  

**提示**：  
- 如果上传的是横版实机录屏，则无需上传此图片。  
- 需与对应端的宣传图 16:9 保持统一。

![gameadmin](https://img.tapimg.com/market/images/3051e795d9d45bd1a0f849a4e074dc03.png)

#### 3.3 宣传片视频

- 用途：首页推荐展示素材  
- 尺寸：1280 x 720px 及以上，推荐 1920 x 1080px  
- 格式：MP4  
- 大小：500 MB 以内  
- 内容：展示游戏的核心玩法为主  

运营建议：游戏更新后可呈现新版本/新角色/新玩法素材、节日相关素材等；视频前 3s 请避免Logo或黑屏。

**提示**：  
首页推荐展示素材可选择：宣传图 16:9、游戏实机录屏、宣传片视频。

![gameadmin](https://img.tapimg.com/market/images/a11103644df7d0537ce4d4ff23d838cb.png)

### 4. 首页编辑推荐栏目推荐图

- 用途：在 TapTap 首页编辑推荐栏目展示的图片，仅限[编辑推荐游戏](/store/operation/advanced-guide/exposure/editorschoice/)  
- 尺寸：1920 x 1280px（3:2）  
- 格式：建议提供 .psd 原文件，必须提供 .png 或 .jpg 图片  

**提示**：  
- 编辑推荐栏目推荐图有更高标准，请详细阅读素材要求。  
- 提交或更新素材请通过[开发者中心](https://developer.taptap.cn/) >> 工单中对应分类提交申请。  
- 如无 .psd 原文件，可能会被拒绝。  
- 非编辑推荐游戏提交素材可能会被处罚。
