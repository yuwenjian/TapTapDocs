## 一、包体收录规范

TapTap 目前支持开发者上传 PC 纯净包，即不含本身的游戏启动器以及其他第三方启动器。

纯净包定义（以下 2 种均可）：

- 打开游戏，进入游戏后，游戏内可以有热更，但是没有「下载/安装/获取」文案  
- 打开启动界面，无需再下载/安装游戏，可以直接启动游戏，游戏内可以有热更，但是没有「下载/安装/获取」文案

游戏包体上传规范：

| **包体类型**     | **包体表现**                                                                                                   | **包体通常大小（仅供参考）** | **收录标准**                                                  |
|------------------|----------------------------------------------------------------------------------------------------------------|-----------------------------|---------------------------------------------------------------|
| **启动器安装包** | 打开启动器安装界面 A，界面有「下载 / 安装 / 更新 / 获取」等文案<br/>点击界面 A 的按钮下载安装<br/>再次打开出现启动界面 B，界面有「下载 / 安装 / 更新 / 获取」等文案<br/>点击界面 B 的按钮下载安装<br/>进入游戏 | 50M 以下                    | 不支持收录                                                    |
| **启动器**       | 打开出现启动界面，界面有「下载 / 安装 / 更新 / 获取」等文案<br/>点击界面的按钮下载安装<br/>进入游戏                                                                 | 50M 到 300M                 | 不建议上传，若无法剥离启动器，请先与对接运营/商务沟通后再传包 |
| **纯净包**       | 【情况1】打开游戏，进入游戏后，游戏内可以有热更，但是没有「下载/安装/获取」文案<br/>【情况2】打开启动界面，无需再下载/安装游戏，可以直接启动游戏，游戏内可以有热更，但是没有「下载/安装/获取」文案 | 通常比较大                  | 推荐上传                                                      |

## 二、如何打包

**步骤一：准备压缩包**（TapTap 后台支持上传 7z、zip 包，大小不限）

在 Windows 上，将游戏文件（通常是包含多个文件或文件夹的整个游戏目录）压缩成一个压缩包（.zip 或 .7z 二选一）

不要解压这个压缩包，接下来整个过程都在压缩包内操作。

**步骤二：浏览压缩包**

使用常用的压缩软件（如 7-Zip, WinZip 等）打开这个游戏压缩包，并浏览里面的内容。

**步骤三：定位启动程序**

在压缩包内浏览，找到游戏真正的启动程序。这通常是一个 .exe 文件，文件名可能包含游戏名称、Start 等字样，或者就是游戏主程序的名字（如 TapTapgame.exe）。

右键点击这个 .exe 文件。

**步骤四：查看文件属性**

在右键菜单中选择 属性，打开该文件的属性窗口。

**步骤五：拼接完整相对路径**

## 三、如何传包

前往开发者中心 >> 商店 >> PC 包体管理 >> 包体管理，点击「上传包体」，选择包体文件，并填写启动项配置。

> 如果您的游戏有依赖库，请同时将依赖库的安装文件放置于游戏目录内一并上传，并在 PC 包体管理 >> 基础配置选择对应的库及库的启动项配置，TapTap 将在用户点击下载安装游戏时，帮助游戏完成依赖库的安装。
