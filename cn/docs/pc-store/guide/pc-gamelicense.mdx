# PC 游戏资质指引

## 一、一张图看懂不同时期的游戏资质

![gameadmin](https://img.tapimg.com/market/images/cc35796152d723ef75dbdaac0af83e20.png)

> **提示**
> 
> 您可使用自研的防沉迷系统，也可使用 TapTap 提供的防沉迷能力。目前平台提供了两种防沉迷合规的能力：
> 
> <b>快捷方案：</b>仅接入 SDK 的启动校验，游戏不提供防沉迷系统。平台将视作游戏无防沉迷系统，拦截未成年人用户的下载和购买。
> 
> <b>完整方案：</b>接入 SDK 的启动校验和合规认证，游戏内自行实现防沉迷系统。平台将视作游戏有防沉迷系统，对所有年龄段的用户开放下载和购买。
> 
> 详细 SDK 接入请查看[《TapTap PC 端集成 Unity 版》](/sdk/integration-guides/pc-engine-features/)、[《TapTap PC 端集成 C++ 版》](/pc/cpp-sdk/overview/)。

**上传资质并提交审核**

开发者资质准备好后，可以前往开发者中心 >> 商店 >> 游戏资料 >> 游戏资质上传相对应的资质，所有资质材料上传完成后，点击页面右上角的「提交审核」按钮，如果已提交的信息需要再次修改，可点击「撤销审核」撤回审核。
