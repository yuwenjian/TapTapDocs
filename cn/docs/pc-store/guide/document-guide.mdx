本篇文档为 TapTap 开发者介绍 PC 游戏如何上架，如何正确使用后台功能，帮助开发者快速了解平台上架，助力游戏发行。


## 一、页面搭建

开发者完成入驻流程后，可以前往开发者后台进行创建游戏，创建游戏时，请先确认游戏符合哪种上架方式。

TapTap 支持两种上架方式：

- 创建同一游戏页，共享资料、评分、论坛。
- 创建两个独立的游戏页，资料、评分、论坛分开维护

具体应该如何选择，请见下图：

![gameadmin](https://img.tapimg.com/market/images/3afa33a8b0bb5d7e11875356907d6ce2.png)

在了解游戏属于哪种上架方式后，请按照以下操作完成类型选项：

| **电脑端和移动端同一游戏页**                          | **电脑端和移动端不同游戏页**                          |
|-------------------------------------------------------|-------------------------------------------------------|
| 请同时勾选「手机+电脑/主机」并按照物料规范上传相关物料。 | 请仅勾选「电脑/主机」并按照物料规范上传相关物料。       |
| ![gameadmin](https://img.tapimg.com/market/images/0d4daddbba949bbf81cac552585c2a20.png)                       | ![gameadmin](https://img.tapimg.com/market/images/4f808b1de3ef8aebaeadc9f031ece1ef.png)                       |

## 四、正式上线

若游戏已准备好面向所有玩家，开启正式上线状态时，开发者可以在商店 >> 游戏资料 >> 商店资料 中修改游戏状态为「正式上线」并提交审核。

**如何开启定时功能**

通过创建游戏和点击「+新版本」时填写的信息，都可在商店 >> 游戏资料>> 商店资料>>发布设置中选择「定时上线」或「立即上线」。定时功能可满足以下几种场景：首曝/首测/首发/版更、发布状态变更、新游戏资料（APK 和素材）。

![gameadmin](https://img.tapimg.com/market/images/83c52f009c8de25a5f7adcc72f9b1ec0.png)

**如何开启预下载功能**

在勾选「正式上线」后，下方会提示您是否要勾选预下载状态，开放预下载后，用户可点击PC端预下载按钮，下载完整安装包，开发者按照游戏需要进行选择即可。

![gameadmin](https://img.tapimg.com/market/images/1da2c537a22ee16cb86e782c8774f6b2.png)

**上线前需要接入的 SDK**

正式上线 TapTap PC 需要您完成接入「启动校验」SDK，其他的登录、防沉迷等 SDK 开发者按需接入即可。

我们建议在首发前2周完成接入，并上传后台提交审核，我们的工作人员会协助您测试接入状态。

**点击下方查看 sdk 接入文档：**

Unity：https://developer.taptap.cn/docs/sdk/access/pc-quickstart/

C++：https://developer.taptap.cn/docs/pc/cpp-sdk/overview/

## 五、游戏更新

当游戏需要更新时，请选择商店，点击当前版本后，会出现一个「+」按钮，点击后即可新建版本页进行更新资料。

![gameadmin](https://img.tapimg.com/market/images/ca6df7caf878ae9383bf56928110a695.png)

**特别注意**

点击「+新版本」时默认复制上一个版本提交的所有信息。

在无需更新包体的情况下，可以直接修改文字或图片素材并提交审核。

如果需要更新包体，请先在PC包体管理上传本次需要上线的包体完成自测，自测无误后在商店页选择「未发布」页面，在游戏下载和获取页面上点击选择历史包体按钮，然后选择需要本次需要上线的包体，最后提交审核。

**TapTap目前提供两种更新方式： 增量更新 & 差量更新**

**增量更新**

基于文件块（Chunk）的更新方式，将文件分割为独立的数据块，更新时只传输变更的数据块

**包体解析**成功即支持，适合开发者自测、回流用户跨版本更新，用户当前任意版本都能高效更新到最新版本

**差量更新**

基于二进制差量算法，精确识别文件二进制变更部分，生成最小化的补丁文件

**提审**通过后支持，适合活跃用户日常更新，更新量最小，为用户节省下载时间和流量

详细规则请阅读：[游戏版本更新规则 | TapTap 开发者文档](https://developer.taptap.cn/docs/pc/update-rules/#%E5%A2%9E%E9%87%8F%E5%B7%AE%E9%87%8F%E6%9B%B4%E6%96%B0%E6%B3%A8%E6%84%8F%E4%BA%8B%E9%A1%B9)

## 六、常见的问题 QA

**如果上传纯净包，TapTap PC 版启动游戏，是否需要提供一些接口支持？**

开发者后台上传纯净包即可正常启动。为了给用户更好的体验，建议开发者接入 TapTap PC 端 SDK 完成启动校验、登录、合规认证等服务接入，帮助玩家规范启动游戏流程，提升游戏转化率。接入文档可查看 [TapTap PC 端集成](https://developer.taptap.cn/docs/sdk/access/pc-quickstart/)。

**关于埋点，是否可以区分用户渠道是 Tap 渠道？**

可以的，我们目前支持自定义启动参数，可以通过启动参数来区分渠道。

**TapTap PC 版是否分成？**

免费下载游戏不分成，付费下载游戏请联系对接商务。

**遇到相关 PC 上架/TapTap PC 版等问题应该怎么办？**

您可以联系对接运营，或者走工单选择分类：商店运营服务支持/TapTap 功能申请/咨询/PC专区咨询。

**TapTap 是所有的游戏都收录吗？**

详情可参考[TapTap 游戏审核规范细则](/store/release/publish/agree/#%E6%9A%82%E4%B8%8D%E6%94%B6%E5%BD%95%E6%B8%B8%E6%88%8F/)。

**创建游戏审核通过后，可暂不发布游戏吗？**

可以，请在商店 >> 游戏资料>> 商店资料>>发布设置中选择「定时上线」然后提交审核。

**我的游戏已经被 TapTap 收录，可以进行游戏认领吗？**

请于[开发者中心](https://developer.taptap.cn/) >> 工单中对应的分类下提交申请，并提供游戏资质材料/企业营业执照扫描件/其他相关证明材料。

**【申请格式 】**

| 申请标题：TapTap 游戏认领申请 - XXX 游戏名称                         |
|----------------------------------------------------------------------|
| 需认领游戏名称及页面链接：                                           |
| 申请认领的厂商名称及厂商页链接：                                     |
| 游戏资质材料（附件上传）：                                           |
| 营业执照扫描件或相关证明材料（附件上传）：                           |
| 申请人联系方式：                                                     |

TapTap 官方人员确认所提交资料合规后，预计在 1-3 个工作日处理。

**游戏主体可以进行转移吗？**

可以。 请于[开发者中心](https://developer.taptap.cn/) >> 工单中对应的分类下提交申请，并提供游戏原所有方厂商和游戏主体新所有方厂商的授权证明材料。

**【 申请格式 】**

| 申请标题：TapTap 游戏主体转移申请 - XXX 游戏名称                     |
|----------------------------------------------------------------------|
| 需进行主体转移的游戏链接：（如有测试服页面必填）                     |
| 游戏主体原所有方厂商名称或厂商页链接：                               |
| 游戏主体新所有方厂商名称或厂商页链接：                               |
| 转让授权书/资质证明（附件上传）：                                    |
| 申请人联系方式：                                                     |

请确保授权链独家且完整，TapTap 官方人员确认所提交资料合规后，预计在 1-5 个工作日内处理。

**如何下架游戏详情页？**

请于[开发者中心](https://developer.taptap.cn/) >> 工单中对应的分类下提交申请，并提供企业营业执照扫描件或其他能证明厂商官方人员的材料。

**【 申请格式 】**

| 申请标题：TapTap 游戏下架申请 - XXX 游戏名称                         |
|----------------------------------------------------------------------|
| 需下架页面的游戏链接：                                               |
| 下架原因：                                                           |
| 期望下架时间：                                                       |
| 营业执照扫描件或资质证明材料（附件上传）：                           |

**特别注意**

下架游戏详情页前，请确保游戏状态为「预约」或「敬请期待」，然后说明下架原因。TapTap 官方人员确认信息后，预计在 1-2 个工作日内处理。

**游戏标题可以修改吗？**

未获取版号/软著/ICP 备案等资质的游戏请在商店 >> 商店资料中点击「+新版本」修改标题然后提交审核。

有已上传版号/软著/ICP 备案等资质的游戏无法直接修改游戏标题，请于[开发者中心](https://developer.taptap.cn/) >> 工单中对应的分类下提交申请，然后在商店 >> 游戏资质中将更名后的资质一并提交审核，同时在商店 >> 商店资料中点击「+新版本」修改标题然后提交审核。

**特别注意**

修改游戏标题时，请统一修改游戏图标、宣传图、游戏截图等资料，游戏标题必须与相关资质全部保持一致。我们建议游戏修改标题后在「开发者的话」或论坛中通知玩家修改游戏标题的原因，以免给玩家带来不良游戏体验，影响游戏评分。
