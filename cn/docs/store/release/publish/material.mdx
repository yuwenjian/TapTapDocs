## 二、 电脑/主机封面及游戏 LOGO、游戏库背景壁纸

### 2.1 游戏 LOGO

- 尺寸：游戏 LOGO 素材尺寸需满足「1280px 宽」或「720px 高」其一即可，在此基础上可以自定义宽高比。
- 格式：png/jpg
- 运营建议：LOGO 内元素请尽量贴合实际尺寸，不用留过大边距，且注意非设计元素以外的空间皆用透明度处理；建议使用最具识别度、和游戏库背景图对比度较大的 LOGO 版本。

![](https://img.tapimg.com/market/images/38c03c28745f3af992b399d41436db9c.png)


### 2.2 横版游戏封面图

* 尺寸：460 x 215px
* 格式：png/jpg
* 图片大小：3 MB 以内
* 运营建议：此图片素材作为游戏的识别名片，需要清晰地展现游戏名和有特点的游戏宣传内容；一般情况下，此图片素材不会作为推广使用，无需跟随游戏版本更新而频繁更换。

![](https://img.tapimg.com/market/images/58a9e7ea4deb8233fa4bdabf8e0749f6.png)


### 2.3 竖版游戏封面图

* 尺寸：600 x 900px
* 格式：png/jpg
* 图片大小：3 MB 以内
* 运营建议：此图片素材作为游戏的识别名片，请保证游戏 LOGO 的清晰和视觉上的高对比度，尽量使用大尺寸的 LOGO 排版方案；背景素材也最好使用具有游戏代表性的视觉元素设计。

![](https://img.tapimg.com/market/images/036d431b56f2d0470aa315a96081cb2c.png)

### 2.4 游戏库背景壁纸

* 尺寸：1920 x 1080px
* 格式：png/jpg
* 图片大小：3 MB 以内
* 注意：避免出现游戏 LOGO、宣传标题等任何形式的文字内容。作为 TapTap PC 版游戏场景下的大图壁纸，请保证图片质量高清且精美。
* 运营建议：游戏库场景下，厂商可以自由更新壁纸和置顶重要的运营内容；为了保证玩家的最佳体验，在壁纸设计时建议尽量考虑到页面左侧有可能出现的公告与活动 UI，避免出现 UI 卡片遮挡背景画面内的主体角色或重要元素；公告与活动 UI 会根据配置情况覆盖在壁纸上方的左下角位置，并根据应用窗口尺寸自适应调整尺寸，但最大宽度不会超过 400px；游戏 LOGO 素材会自动布局在画面右上角，建议 LOGO 下方使用简单干净、能突显 LOGO 颜色和细节的背景设计。

![](https://img.tapimg.com/market/images/1a0f9f0a4a8ff1196477192a7c13a2a0.png)

## 三、 文案

### 3.1 简介 <sup><Red>\* 必填</Red></sup>

- 内容：描述游戏的类型、玩法以及特色。
- 运营建议：建议将主要内容控制在前 50 个字，有利于加深用户对游戏的第一印象。

### 3.2 开发者的话

- 内容：描述游戏设计理念、游戏开发心路历程。
- 运营建议：建议开发者的话具备个人特色且真情实感，并建议将主要内容控制在前 50 个字。为避免影响用户端展示效果，简介与开发者的话请勿雷同。

### 3.3 更新日志 <sup><Red>\* 必填</Red></sup>

- 内容：描述本次游戏更新内容与游戏版本变动，不得出现宣传、广告、联系方式或其他无关内容。
- 运营建议：建议文案简洁明了，使用有序列表展示更新点，比如：1. 2. 3...

![](https://img.tapimg.com/market/images/54ce9efb73a49cd8f9c372f04a717d05.png)

## 四、 实机录屏及游戏截图

### 4.1 游戏实机录屏 <sup><Red>\* 下载/试玩必填</Red></sup>

- 格式：MP4
- 大小：500 MB 以内
- 长度：视频长度最短大于 15 秒，最长不超过 30 分钟。
- 内容：视频需演示游戏的核心玩法。
- 规范：

1. 为保证用户能够快速了解游戏玩法，游戏正式上线开放下载或开放试玩版必须提供实机视频。
2. 请在前 5 秒内展示核心玩法且保持连续录制超过 10 秒，前 30 秒请勿出现超过 10 秒的纯展示、CG、立绘、抽卡动画、过场动画；建议不要从头开始录制，直接录最核心、玩法最密集的画面；避免使用抽卡动画 / 过场动画等类似 pv 的画面，占据视频的主要时长，且请避免仅用较短时间演示核心玩法，以掩盖游戏本身的真实水平、误导玩家。
3. 请勿添加游戏本体以外的任何配音和字幕，可适当剪辑视频（如加速/减速/拼接）以提升观看体验。
4. 请勿出现任何形式的前贴片，请勿出现除厂商 logo 和游戏 logo 以外形式的后贴片
5. 请勿包含其他平台LOGO水印，请勿在视频中直接出现其他三方平台导流内容。
6. 请勿使用圣光/暗幕/云团/表情包/马赛克等方式进行遮挡画面。
7. 请勿使用画中画，请确保录屏内容占比在 80% 以上。
8. 请勿拍出游玩设备，请使用游戏本身的录屏画面。
9. 请保持视频长度最短大于 15 秒，最长不超过 30 分钟。
10. 请勿含有营销诱导性文案。
11. 请提供高清视频。

![](https://img.tapimg.com/market/images/5054ab3afd7b78dfb7d7cfd9da90bee6.png)

### 4.2 游戏截图<sup><Red>\* 必填</Red></sup>

- 图片方向：支持选择横图、竖图方式展示游戏画面，但请保证每张截图的图片方向一致。如果选择上传横图，请保证所有的图片均为横图
- 尺寸：横版 1280 x 720px 及以上；竖版 720 x 1280px 及以上
- 比例规范：大部分手机截图均可直接上传。横图宽高比需在 8:3 到 8:5 之间。竖图宽高比需在 3:8 到 5:8 之间。
- 格式：png/jpg
- 数量：至少 3 张，请勿上传相同截图
- 大小：建议每张截图 1 MB以内，截图文件大小过大会导致上传速度及用户打开游戏详情页时加载速度过慢，影响游戏转化
- 注意：为保证用户能够快速了解游戏玩法，在 TapTap 开放包体下载、测试的游戏截图中必须有实机核心玩法内容展示，实机画面至少占图片的 50%，概念类、广告宣传类图片至多 2 张；游戏截图使用游戏中不存在的画面、过度美化游戏中的画面、仅展示卡片、立绘、Loading 界面、主界面等行为，以掩盖游戏本身的真实美术水平、核心玩法，对用户造成误导的，都可能对游戏的正常分发造成影响。

![](https://img.tapimg.com/market/images/9449bd4adc323b33b6b3ae494a6090ef.png)

## 五、推广物料

### 5.1 宣传图 16:9 <sup><Red>\* 必填</Red></sup>

- 用途：首页推荐、今日游戏等使用卡片展示游戏的页面，用于宣传游戏；在游戏预约等阶段，未上传游戏实机录屏的情况下，作为游戏详情页的首张图片，用以突出游戏的品牌形象；同时用于横版游戏实机录屏、宣传视频（如有）的视频封面。
- 尺寸：1920 x 1080px（16:9）
- 格式：png/jpg
- 内容：建议使用游戏主要角色、主要画面场景作为图片的核心视觉，且图片上必须展示清晰可见的游戏名
- 规范：

1. 除了游戏名外，请勿引用或使用其他文本。
2. 请保证宣传图美观，不符合 TapTap 平台调性的宣传图将会被审核驳回。
3. 在某些页面会使用 4:3 比例对图片做裁剪进行展示，请勿将游戏 logo、主要角色放置在裁剪线上。
4. 请勿使用游戏标题在图内做大字报宣传，并且纯色底/渐变色的宣传图请勿仅有游戏标题，请添加其他装饰元素。

![](https://img.tapimg.com/market/images/1ec50b2b6616a7037d86bf01ec6a9dc3.png)

### 5.2 宣传图 1:1

- 用途：竖版实机录屏的视频封面
- 尺寸：1440 x 1440px（1:1）
- 格式：png/jpg
- 内容：除游戏名的位置和大小、游戏主要角色的位置和大小等为了适配当前尺寸而调整的细节外，请保证图片主体内容与宣传图 (16:9) 一致
- 规范：

1. 如果上传了横版实机录屏而非竖版，则无需上传此图片。
2. 宣传图 1:1 的主体内容请与宣传图 16:9 保持统一，并且安卓端的宣传图 1:1 的主体内容需要与安卓端宣传图 16:9 保持统一，iOS 端的宣传图 1:1 的主体内容需要与 iOS 端宣传图 16:9 保持统一。

![](https://img.tapimg.com/market/images/b38a131959b731454a9e9b87ff2fa7a7.png)


### 5.3 宣传片

- 用途：游戏在首页推荐展示素材
- 尺寸：1280 x 720px 及以上，推荐 1920 x 1080px
- 格式：MP4
- 大小：500 MB 以内
- 内容：展示游戏的核心玩法为主
- 运营建议：游戏更新后可呈现游戏新版本/新角色/新玩法素材、节日相关素材等；视频前 3s 代表了游戏的第一印象，请避免开屏的 logo、黑屏等无游戏实机画面的情况。
- 提示：请按需设置游戏在首页推荐展示效果，可选择的推荐素材有：宣传图 16:9、游戏实机录屏、宣传片视频。

![](https://img.tapimg.com/market/images/a11103644df7d0537ce4d4ff23d838cb.png)

## 六、 首页编辑推荐栏目推荐图

- 用途：在 TapTap 首页编辑推荐栏目展示的图片，仅限 [编辑推荐游戏](https://developer.taptap.cn/docs/store/store-editorschoice/)
- 尺寸：1920 x 1280px（3:2）
- 格式：建议提供 .psd 原文件，必须提供 .png 或 .jpg 图片
- 提示：

1. 相较于其他物料，编辑推荐栏目推荐图有更高标准，请详细阅读素材要求。提交新素材，或更新素材，请通过[开发者中心](https://developer.taptap.cn/) >> 工单中对应分类提交申请。
2. 当图片不满足编辑推荐位的素材要求时，TapTap 将优先尝试使用 .psd 原文件重新制作图片。如无 .psd 原文件可使用时，将直接拒绝游戏素材。
3. 如果游戏暂未提供素材，TapTap 将按游戏现有素材重新制作推荐图。如果重新制作推荐图的游戏数量较多，TapTap 将按照一定次序为每款游戏排期制作。
4. 没有首页编辑推荐栏目推荐图的游戏将不展示在编辑推荐游戏的专属栏目。
5. 当前推荐图仅限编辑推荐的游戏，非编辑推荐的游戏提交素材，可能会被判定影响平台运营工作，并受到相应的处罚。
