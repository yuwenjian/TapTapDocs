## 三、 文案

游戏如果需要上架中国大陆地区，请提供简体中文版文案与物料。
文案里请勿包含兑换码（包括通码）、玩家交流群信息、除 Steam 平台和 TapTap 详情页链接之外的外部链接。

### 1. 简介 <sup><Red>\* 必填</Red></sup>

1.1 内容：描述游戏的类型、玩法以及特色

1.2 运营建议：建议将主要内容控制在前 50 个字，有利于加深用户对游戏的第一印象

### 2. 开发者的话

2.1 内容：描述游戏设计理念、游戏开发心路历程

2.2 运营建议：建议开发者的话具备个人特色且真情实感，并建议将主要内容控制在前 50 个字

:::tip

为避免影响用户端展示效果，简介与开发者的话请勿雷同。

:::

### 3. 更新日志

3.1 内容：描述本次游戏更新内容与游戏版本变动，不得出现宣传、广告、联系方式或其他无关内容

3.2 运营建议：建议文案简洁明了，使用有序列表展示更新点，比如：1. 2. 3...

![文案](https://img.tapimg.com/market/images/beeefd15accf59c22e60b9be849eb59b.jpg)

## 四、 实机录屏及游戏截图

### 1. 游戏实机录屏<sup><Red>\* 可下载或可试玩必填</Red></sup>

1.1 格式：MP4  

1.2 大小：500 MB 以内  

1.3 长度：视频长度建议在 2 分钟内，最长不得超过 3 分钟

1.4 内容：视频需演示游戏的核心玩法

:::tip

1. 为保证用户能够快速了解游戏玩法，游戏正式上线开放下载或开放试玩版必须提供实机视频。
2. 请在前 5s 内展示核心玩法，建议不要从头开始录制，直接录最核心、玩法最密集的画面。请避免使用抽卡动画 / 过场动画等类似 pv 的画面，占据视频的主要时长，且请避免仅用较短时间演示核心玩法，以掩盖游戏本身的真实水平、误导玩家。
3. 请勿只展示角色、立绘、Loading 界面、主界面等画面，需演示游戏的核心玩法（建议占比在 90% 以上）。
4. 请勿添加游戏本体以外的任何配音和字幕，可适当剪辑视频（如加速/减速/拼接）以提升观看体验。
5. 前 30 秒内请勿在全屏画面中只展示游戏/厂商/引擎 Logo。
6. 请勿额外使用人物立绘，请确保录屏内容占比在 80% 以上。
7. 请勿使用画中画，请确保录屏内容占比在 80% 以上。
8. 请勿拍出游玩设备，请使用游戏本身的录屏画面。
9. 请保持在 2 分钟以内，最长请勿超过 3 分钟。
10. 片尾请勿含有营销诱导性文案。
11. 请提供高清视频。

:::

![实机录屏](https://img.tapimg.com/market/images/1abf410530989f75e85636d09da3b825.jpg)


### 2. 截图<sup><Red>\* 必填</Red></sup>

2.1 图片方向：支持选择横图、竖图方式展示游戏画面，但请保证每张截图的图片方向一致。如果选择上传横图，请保证所有的图片均为横图

2.2 尺寸：横版 1280 x 720px 及以上；竖版 720 x 1280px 及以上

2.3 格式：png/jpg

2.4 数量：至少 3 张，请勿上传相同截图

2.5 大小：建议每张截图 1 MB以内，截图文件大小过大会导致上传速度及用户打开游戏详情页时加载速度过慢，影响游戏转化

:::tip

1. 为保证用户能够快速了解游戏玩法，在 TapTap 开放包体下载、测试的游戏截图中必须有实机核心玩法内容展示，实机画面至少占图片的 50%，概念类、广告宣传类图片至多 2 张。
2. 游戏截图使用游戏中不存在的画面、过度美化游戏中的画面、仅展示卡片、立绘、Loading 界面、主界面等行为，以掩盖游戏本身的真实美术水平、核心玩法，对用户造成误导的，都可能对游戏的正常分发造成影响。

:::

![截图](https://img.tapimg.com/market/images/6662a2233bb3ae75fe3933558dce11ad.jpg)

## 五、推广物料

### 1. **宣传图 16:9 **<sup><Red>\* 必填</Red></sup>

1.1 用途：首页推荐、今日游戏等使用卡片展示游戏的页面，用于宣传游戏；在游戏预约等阶段，未上传游戏实机录屏的情况下，作为游戏详情页的首张图片，用以突出游戏的品牌形象；同时用于横版游戏实机录屏、宣传视频（如有）的视频封面

1.2 尺寸：1920 x 1080px（16:9）

1.3 格式：png/jpg

1.4 内容：建议使用游戏主要角色、主要画面场景作为图片的核心视觉，且图片上必须展示清晰可见的游戏名

:::tip

1. 除了游戏名外，请勿引用或使用其他文本。
2. 请保证宣传图美观，不符合 TapTap 平台调性的宣传图将会被审核驳回。
3. 在某些页面会使用 4:3 比例对图片做裁剪进行展示，请勿将游戏 logo、主要角色放置在裁剪线上。
4. 请勿使用游戏标题在图内做大字报宣传，并且纯色底/渐变色的宣传图请勿仅有游戏标题，请添加其他装饰元素。

:::

![宣传图 16:9](https://img.tapimg.com/market/images/4ce6923195a69a8261ca1ab8c0a18dc8.jpg)

### 2. **宣传图 1:1 **

2.1 用途：竖版实机录屏的视频封面

2.2 尺寸：1440 x 1440px（1:1）

2.3 格式：png/jpg

2.4 内容：除游戏名的位置和大小、游戏主要角色的位置和大小等为了适配当前尺寸而调整的细节外，请保证图片主体内容与宣传图 (16:9) 一致

:::tip

1. 如果上传了横版实机录屏而非竖版，则无需上传此图片。
2. 宣传图 1:1 的主体内容请与宣传图 16:9 保持统一，并且安卓端的宣传图 1:1 的主体内容需要与安卓端宣传图 16:9 保持统一，iOS 端的宣传图 1:1 的主体内容需要与 iOS 端宣传图 16:9 保持统一。

:::

![宣传图 1:1](https://img.tapimg.com/market/images/d766ecbc14d7456d7286a4ee6d7b9d24.jpg)


### 3. 宣传片视频

3.1 用途：游戏在首页推荐展示素材

3.2 尺寸：1280 x 720px 及以上，推荐 1920 x 1080px 

3.3 格式：MP4

3.4 大小：500 MB 以内

3.5 内容：展示游戏的核心玩法为主

3.6 运营建议：游戏更新后可呈现游戏新版本/新角色/新玩法素材、节日相关素材等；视频前 3s 代表了游戏的第一印象，请避免开屏的 logo、黑屏等无游戏实机画面的情况

:::tip

请按需设置游戏在首页推荐展示效果，可选择的推荐素材有：宣传图 16:9、游戏实机录屏、宣传片视频。

:::

![宣传片视频](https://img.tapimg.com/market/images/61381f12ac0271f75f1574c8acdd9323.jpg)


## 六、 首页编辑推荐栏目推荐图

1.1 用途：在 TapTap 首页编辑推荐栏目展示的图片，仅限[编辑推荐游戏](/store/store-editorschoice/)

1.2 尺寸：1920 x 1280px（3:2）

1.3 格式：建议提供 .psd 原文件，必须提供 .png 或 .jpg 图片

:::tip

1. 相较于其他物料，编辑推荐栏目推荐图有更高标准，请详细阅读素材要求。提交新素材，或更新素材，请通过[开发者中心](https://developer.taptap.cn/) >> <Blue>工单</Blue>中对应分类提交申请。
2. 当图片不满足编辑推荐位的素材要求时，TapTap 将优先尝试使用 .psd 原文件重新制作图片。如无 .psd 原文件可使用时，将直接拒绝游戏素材。
3. 如果游戏暂未提供素材，TapTap 将按游戏现有素材重新制作推荐图。如果重新制作推荐图的游戏数量较多，TapTap 将按照一定次序为每款游戏排期制作。
4. 没有首页编辑推荐栏目推荐图的游戏将不展示在编辑推荐游戏的专属栏目。
5. 当前推荐图仅限编辑推荐的游戏，非编辑推荐的游戏提交素材，可能会被判定影响平台运营工作，并受到相应的处罚。

:::

![编辑推荐栏目推荐图](https://img.tapimg.com/market/images/f5a74308b25853219aeaccb094aaf3c6.jpg)
