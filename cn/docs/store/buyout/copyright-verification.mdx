# 买断制游戏

TapTap 为买断制游戏提供完整上架服务，包括 SDK 接入、价格折扣管理、优惠券营销功能、销售数据监控和订单结算。

![](https://img.tapimg.com/market/images/e0582de465379e63ba2df5ad3937de4a.png)

## 支持多种付费解锁形式

根据游戏的销售策略，您可以使用 TapTap 的服务实现以下任意付费解锁形式：

* **付费下载**：用户需要付费才可以下载游戏包体，并且启动游戏时需校验购买资格，用户使用非法途径获取游戏包体将无法进入游戏。**TapTap 对于此类付费下载游戏有额外的流量扶持**。
* **付费下载、付费解锁更多内容**：用户付费下载游戏包体后可以游玩全部内容，但还需付费才可解锁更多曲包、皮肤、附加章节等额外内容。
* **免费下载、付费解锁完整版**：用户可免费下载游戏包体，并根据游戏设置试玩少量内容，但还需付费才可解锁完整版。用户能够试玩内容量的多少由游戏通过代码自行控制。
* **免费下载、付费解锁更多内容**：用户可免费下载游戏包体，但还需付费才可解锁更多曲包、皮肤、附加章节等额外内容。

## 如何上架买断制游戏

### 前期准备

1. 创建游戏，补充游戏资质，并提交审核。创建游戏时，**请务必填写正确的游戏包名。**
2. 审核通过后，在开发者中心 >> 游戏服务 >> 正版验证功能，开通正版验证服务。
3. 如游戏需接入付费下载，请直接查阅  [开发指南](/sdk/copyright-verification/guide/) ，完成正版验证 SDK 的研发接入并自测通过。
4. 如游戏还需使用付费解锁更多内容、付费解锁完整版功能，请先查阅  [开发指南](/sdk/copyright-verification/guide/) ，根据文档指引创建 [DLC 商品](/store/buyout/dlc-products/) ，完成正版验证 SDK 的研发接入并自测通过。

### 正式上线

1. 将游戏包体提交审核，设置定时上线时间。
2. 前往厂商设置 >> 财务与管理 >> 财务主体功能，设置财务主体，该财务主体将用于收款。
3. 在开发者中心 >> 商店 >> 买断制游戏功能，开启付费下载并设置价格。该页面也可设置首发折扣。

## 如何结算

TapTap 与游戏的结算周期为月结，结算方式为对公打款。您可前往厂商设置 >> 财务与管理 >> 财务对账功能查看已生成的对账单，并确认对账单后联系 TapTap 工作人员并完成结算。

## 营销活动

TapTap 支持两种营销方式：

* **限时打折**：在限定时间内面向全部用户打折，最低可设置 0.01 元。您可前往开发者中心 >> 商店 >> 买断制游戏功能设置。
* **发放优惠券**：优惠券仅限少量已获取优惠券的用户可用，详细请查看 [折扣与优惠券](/store/buyout/discounts-coupons/)。需注意，**折扣和优惠券可叠加使用**。

## 常见问题

#### Q：付费下载游戏有额外的流量扶持具体是什么？

付费下载游戏在 TapTap 的各大榜单、推荐算法等自然流量计算时，有一定程度的加权。免费下载的权重同其他非买断制游戏。

#### Q：为什么游戏未计入热卖榜？

热卖榜仅计入付费下载、免费下载付费解锁完整版的游戏。
