## 一、成为 TapTap 开发者

前往 TapTap [开发者中心](https://developer.taptap.cn/) >> <Blue>进入控制台</Blue> >> 选择目标开发者类型 >> <Blue>申请入驻</Blue>。

![gameadmin](https://img.tapimg.com/market/images/54468239fe3f2b64a21d0442658cd522.png)

## 二、开发者类型

TapTap 开发者分为**<Blue>注册开发者</Blue>**和**<Blue>认证开发者</Blue>**，提交入驻申请的 TapTap 账号，将在开发者审核通过后自动成为该开发者账号的超级管理员。

已成为**<Blue>注册开发者</Blue>**账号的厂商主体，可以在**[开发者中心](https://developer.taptap.cn/) >> <Blue>财务与管理</Blue> >> <Blue>厂商设置</Blue> >> <Blue>成为认证开发者</Blue>**位置申请转换开发者类型，无需重复创建厂商申请账号。

| 入驻类型   | 前提条件               | 必要的资料信息                                                                                                                                                                                                                                                                          | 非必要的资料信息                                                                                                                                                                  | 是否需要审核 | 厂商详情页           |
| ---------- | ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------ | -------------------- |
| 注册开发者 | 具备可用的 TapTap 账号 | 厂商名称、邮箱、手机号（需要验证）                                                                                                                                                                                                                                                      | 厂商英文名称                                                                                                                                                                      | 无需审核     | 不对玩家展示         |
| 认证开发者 | 具备可用的 TapTap 账号 | **个人主体** <br/> <ul><li>厂商名称、开发者姓名、常用地址、身份证正反面扫描件</li><li>邮箱、手机号（需要验证）</li></ul> <br/> **企业主体** <br/> <ul><li>厂商名称、单位名称、注册地址、统一社会信用代码、营业执照扫描件、认证公函扫描件</li><li>联系邮箱、手机号（需要验证）</li><li>紧急联系人信息</li></ul> | **个人主体** <br/><ul><li> 厂商英文名称</li><li>QQ、微信</li></ul> <br/>**企业主体**<br/> <ul><li>厂商英文名称</li><li>QQ、微信</li><li>法人信息</li></ul> | 需要审核     | 审核通过后对玩家展示 |

![gameadmin](https://img.tapimg.com/market/images/1ba33c66363dc9bf94bc131d316d4d02.png)

**特别注意**

1. 注册开发者仅支持体验开发者后台部分功能，所创建的游戏和厂商信息都不会对玩家展示。

2. 注册开发者通道最多支持创建 2 个「未发布」状态的游戏，且游戏无法提交审核。如需在 TapTap 平台运营上线游戏，需要选择入驻厂商类型为认证开发者。

3. 注册开发者不区分个人/企业身份，申请成为认证开发者时可选择身份类型。

4. 注册开发者可以申请成为认证开发者，但认证开发者不支持成为注册开发者。

## 五、常见问题 Q&A
### 1. 个人开发者与企业开发者有什么区别？

个人开发者与企业开发者区别仅在于申请开发者时提供的申请资料类型不同，开发者中心的功能体验也无区别。

个人主体需提供开发者个人信息，企业主体则需要提供企业信息。（该信息可在开发者注册完成后在<Blue>厂商设置</Blue> >> <Blue>账号信息</Blue>中查看）

### 2.个人开发者没有公司，如何填写厂商名称？

建议以团队或工作室名称而非个人名称作为厂商名称，例如：XXX 工作室。

### 3. 厂商名称已存在，如何进行厂商账号认领？

请于[开发者中心](https://developer.taptap.cn/) >> <Blue>工单</Blue>中对应的分类下提交申请，或邮件发送至运营部邮箱 operation@taptap.com ，并提供企业营业执照扫描件或其他能证明厂商官方人员的材料。
**申请格式**
> 申请标题：TapTap 厂商账号认领申请 - XXX 厂商名称
> 
> 需认领的厂商名称及厂商页链接：
> 
> 营业执照扫描件或相关证明材料（附件上传） ：
> 
> 申请人联系方式：

TapTap 工作人员确认提交资料后，将在 1-3 个工作日内操作解绑厂商账号绑定的超级管理员，然后厂商可前往[开发者中心](https://developer.taptap.cn/)使用相同的厂商名称创建厂商账号，提交审核。
审核通过后，申请成为厂商的用户默认为超级管理员。

### 4. 如何修改厂商在详情页的展示名称

「厂商名称」创建后无法自行修改，如需修改请于[开发者中心](https://developer.taptap.cn/) >> <Blue>工单</Blue>中对应的分类下提交申请。

### 5. 开发者审核通过后需要立即进行游戏入库吗？

不需要。
TapTap 对创建游戏时间无要求，请厂商根据自己的时间安排进行游戏创建。但 TapTap 建议尽早进行游戏入库，积累预约玩家。

### 6. 厂商主体可以注销不对玩家展示吗？

可以。厂商主体注销是不可逆的操作，请确认需要注销主体后，在[开发者中心](https://developer.taptap.cn/) >> <Blue>工单</Blue>中对应的分类下提交申请。
请注意，厂商账号注销后将无法再次进入厂商账号，如需重新发布游戏请创建新的厂商账号。
