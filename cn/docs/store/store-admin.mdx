## 一、权限管理

**仅超级管理员可进行此项操作（首次申请成为开发者的用户默认为超级管理员）。**

### 1. 添加成员或超级管理员

进入<Blue>权限管理</Blue> >> 点击<Blue>添加用户</Blue> >> 输入用户昵称或用户的 TapID >> 角色类型选择「成员」或「超级管理员」。超级管理员拥有所有厂商权限。

![gameadmin](https://img.tapimg.com/market/images/979ef05d4177b1c664eb5fd8ae853833.png)

### 2. 用户权限设置和删除

用户权限可以划分为「厂商权限」和「游戏权限」。

进入<Blue>权限管理</Blue> >> <Blue>编辑权限</Blue> >> <Blue>厂商权限</Blue> >> 配置厂商权限，可以设置成为「厂商管理员」、「财务」、「全局数据分析」。「厂商管理员」拥有创建游戏、更新游戏资料等操作权限，「财务」拥有管理账单、游戏服务账户的操作权限，「全局数据分析」拥有查看游戏数据，自动获得数据分析服务中「全局观察者」的权限。

![gameadmin](https://img.tapimg.com/market/images/a5085f56c5deeed90d94373d1d56630c.png)
![gameadmin](https://img.tapimg.com/market/images/97786c325e844dc9d194eb01a175d958.png)

进入 <Blue>权限管理</Blue> >> <Blue>编辑权限</Blue> >> <Blue>游戏权限</Blue> >> 配置游戏权限，可以根据实际需要单独管理不同游戏的权限。
![gameadmin](https://img.tapimg.com/market/images/a5085f56c5deeed90d94373d1d56630c.png)
![gameadmin](https://img.tapimg.com/market/images/74aadab291510b8f64091cd3ec90af5b.png)

在对应用户后面点击 “┆” 按钮，可将该用户设为「超级管理员」或移除「用户」。

![gameadmin](https://img.tapimg.com/market/images/d357884af3770e218ecc34166232df75.png)

### 3. 角色配置

<Blue>权限管理</Blue> >> <Blue>角色配置</Blue>中默认配置了一些常用的角色。如：游戏管理员、开发、发行、投放等，对应角色所拥有的权限可在“查看角色”中了解，默认角色无法添加或移除权限。

如需自定义角色权限，可选择「添加角色」。角色类型分两种，一种为针对游戏页管理权限的「游戏角色」，另一种为针对厂商页管理权限的「厂商角色」，主管理员添加角色后根据需要进行命名和权限勾选即可。

![gameadmin](https://img.tapimg.com/market/images/bd514e420178d83bf8bf69b5705cff8d.png)

## 二、工单咨询

TapTap 提供一站式自助功能，如遇到更多问题可在[开发者中心](https://developer.taptap.cn/) >> <Blue>工单</Blue>中创建工单咨询。工单系统入口位于开发者中心顶部导航栏右侧位置。

## 三、开发者认证

TapTap 开发者认证是企业或个人开发者独有的认证，已认证玩家头像下方将会有蓝色“√”标志，该认证头衔会显示在个人主页。此认证仅为展示作用，不包含开发者中心的权限。账号认证后，用户可以在论坛使用定时发帖的功能。

申请和移除开发者认证，可在[开发者中心](https://developer.taptap.cn/) >> <Blue>工单</Blue>中对应的分类下提交申请。头衔固定形式为「厂商名称/游戏名称」+「官方/职位」（申请时请勿使用花名，如吉祥物、打杂等）。

![gameadmin](/img/开发者认证.png)

## 四、常见问题

### **1. 用户账号绑定的手机号或邮箱可以更换吗？**

可以。
请进入<Blue>个人中心</Blue> >> <Blue>帮助与反馈</Blue> >> <Blue>我要反馈</Blue> >> 详细描述遇到的问题。

### **2. 开发者认证可以绑定多个账号吗？**

一个账号最多认证 3 款游戏/开发者信息，同一个游戏主体或厂商账号最多支持 10 个用户账号绑定开发者认证。

### **3. 游戏还没有上架可以申请开发者认证吗？**

不支持，请确保游戏已经上架并且游戏详情页对玩家可见。

### **4. 开发者认证有使用期限吗？**

没有，如果需要移除认证请主动联系 TapTap 工作人员操作。

### **5. 为什么无法查看厂商账号名下提交的其他工单？**

工单的账号系统是独立的，只能看见个人账号提交的信息。
