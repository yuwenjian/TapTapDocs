# 模拟真实文件结构测试用例 - hk

## 域名设置指南

请在初始化客户端 SDK 时传入您的 API 域名，例如：

<Conditional brand="tds">

```java
// 初始化 TDS SDK
TapGameCenter.init(context, appId, clientToken, config -> {
    config.setServerUrl("https://your-api-domain.com");
});
```

在控制台中绑定 API 域名和文件访问域名，并完成相应的 DNS 解析配置。

<Conditional region="global">
在国际区域，域名不需要备案，但建议使用专业的域名注册商服务。
</Conditional>

<Conditional region="cn">
在中国大陆区域，域名需要完成备案才能使用。
</Conditional>

注意：在游戏测试阶段，您可以使用我们提供的共享域名。上线前，务必切换到您自己的已备案域名，以避免潜在的 DDoS 攻击风险。

</Conditional>

<Conditional brand="leancloud">

```java
// 初始化 LeanCloud SDK
AVOSCloud.initialize(context, appId, clientKey, serverUrl);
```

LeanCloud 平台要求绑定自有域名：

<Conditional region="global">
1. 国际版可以使用我们提供的共享域名
2. 如需自有域名，请在控制台设置
</Conditional>

<Conditional region="cn">
1. 在中国大陆节点，必须绑定已备案的域名
2. 请在 LeanCloud 控制台 > 设置 > 域名绑定页面完成设置
</Conditional>

您可以在控制台查看域名配置指南。

</Conditional>
