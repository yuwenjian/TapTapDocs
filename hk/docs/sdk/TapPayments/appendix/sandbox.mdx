# 沙盒环境

TapTap Payments 沙盒环境进行支付测试, 可以使用一些沙盒信用卡进行支付
:::tip
信用卡不支持 $0.5 以下的支付, 测试商品时，请确保商品定价高于 $0.5
:::

## 沙盒账号设置
进入开发者中心后台, 按照如下步骤设置和添加对应沙盒账号:

![](https://img.tapimg.com/market/images/7309efbf7c1971d44c73a53494193bb4.png)

## 沙盒信用卡
在支付时, 可以使用以下沙盒信用卡进行支付测试, 输入任意 CVC（3位数字）、邮政编码、有效期和支付密码的前两位都会被接受:

![](https://img.tapimg.com/market/images/ca74786cdb7410fdf23c9802b4a5527d.png)
